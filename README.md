# crm-app

CRM Android app is designed to be used by Truebil sales representatives at Truebil's sales offices. The app will help the sales representatives to easily view daily appointments as well as to see the past history of customers who visit the center for test drives. The app also includes features to create new appointments, view car details, initiate deal terms and fill payment details.

## Setup ##
Steps for running this app:

1. Clone the app and build and run using Android Studio
2. Create the debug or release apk for distribution

## Pre-requisites ##
1. Android device running API 15 (Android IceCreamSandwich) or later
2. Latest Android Studio for compiling the source code
